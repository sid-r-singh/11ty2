Staart is a set of starter tem­plates to build a SaaS (software-as-a-service) startup us­ing Node.js and TypeScript. It has built-in au­then­ti­ca­tion (with 2FA and OAuth2 con­nect), team man­age­ment with roles and in­vites, re­cur­ring pay­ments with Stripe, an API gate­way with rate lim­it­ing and billing, and more.

Staart UI is built with Vue.js and is the fron­tend pro­ject, while Staart API is the back­end pro­ject. Staart UI also con­tains fea­ture and land­ing pages, like pric­ing and poli­cies.

Staart is free and open-sourced on GitHub.
